## Key Features Demonstrated

1. **3D Interaction**: Drag to rotate, scroll to zoom
2. **Resource Overlays**: Toggle different lunar resources (water ice, helium-3, minerals)
3. **Search & Filter**: Find specific missions like Apollo programs
4. **Visual Settings**: Advanced lighting, starfield, and surface customization
5. **Feature Layers**: Craters, maria, and other lunar features
6. **Status Legend**: Color-coded artifact types (Impactor, Crashed, Landed, Orbiting)
7. **Origin Filters**: Filter by country/space agency
